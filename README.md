# Creator Governance

**Tax compliance and audit trails for creator businesses.**

## Features

- Stripe Connect integration
- Automated GST/HST calculation
- Audit-ready reports
- CRA compliance
- Multi-currency support

## Quick Start

```bash
npm install
npx prisma db push
npm run dev
```

## Tech Stack

- Next.js 14
- Prisma + PostgreSQL
- Stripe Connect
- Tailwind CSS
