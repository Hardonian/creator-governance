# Creator Governance

**Tax compliance and audit trails for creator businesses.**

![License](https://img.shields.io/badge/License-MIT-green)
![TypeScript](https://img.shields.io/badge/TypeScript-5.3-blue)

## Why

Creators selling digital products need:
- GST/HST calculation
- Audit-ready records
- CRA compliance
- Stripe + bank integration

This platform handles it all.

## Features

- 💳 **Stripe Connect** — Real revenue tracking
- 🧮 **Tax Calculation** — Auto GST/HST based on province
- 📊 **Audit Trails** — Timestamped, immutable records
- 📄 **Tax Reports** — CRA-ready PDF exports
- 🔒 **Privacy First** — Your data stays yours

## Quick Start

```bash
npm install
npx prisma db push
npm run dev
```

Visit http://localhost:3000

## Tech Stack

- **Frontend:** Next.js 14, React, Tailwind CSS
- **Backend:** Prisma, PostgreSQL
- **Payments:** Stripe Connect
- **Auth:** Clerk
